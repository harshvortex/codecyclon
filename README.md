## 📦 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/codecyclon.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env.local` file and add your API keys:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Architecture

The project follows a modular architecture with a focus on performance and visual excellence:

- `/src/components`: Reusable UI components and neural background systems.
- `/src/lib`: Utility functions and core logic.
- `/src/App.tsx`: Main entry point and layout orchestration.
